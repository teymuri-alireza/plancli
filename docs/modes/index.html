# What are modes?

plancli supports 2 different modes

• CLI mode (which is written using Spectre.Console)
• TUI mode (which is written using Terminal.Gui)

> CLI mode

plancli --mode cli

This mode uses Spectre Console. List of the tasks is printed as a table and the rest of functions can be chosen with a prompt.

> TUI mode

plancli --mode tui

This mode uses Terminal.Gui. A new TUI window shows up so user can chosse a specific task to edit, and there are commands for adding a new task and for settings.
